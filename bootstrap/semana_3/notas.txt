## JS
(role="tablist", role="tab", role="tabpanel", and additional aria- attributes)

* tabs
  * se crea una lista con clases especificas como: .nav, .nav-tabs, .nav-item, .nav-link y .nav-link.active
  * se debe agregar el id
  * tambien agregarle un rol (role="tablist")
  * Ej:
    `
    <ul class="nav nav-tabs" id="misTabs" role="tablist">
    <li class="nav-item" role="presentation">
      <a class="nav-link" id="home-tab" data-toggle="tab" href="#home" role="tab" aria-controls="home" aria-selected="true">Home</a>      
    </li>
    <li class="nav-item" role="presentation">
      <a class="nav-link" id="profile-tab" data-toggle="tab" href="#profile" role="tab" aria-controls="profile" aria-selected="false">Profile</a>      
    </li>
     <li class="nav-item" role="presentation">
      <a class="nav-link" id="contact-tab" data-toggle="tab" href="#contact" role="tab" aria-controls="contact" aria-selected="false">Contact</a>      
    </li>
    </ul>
    <div class="tab-content" id="miTabContent">
      <div class="tab-pane fade show active" id="home" role="tabpanel" aria-labelledby="home-tab">
      <p>Hola pestaña HOME</p>
      </div>
      <div class="tab-pane fade" id="profile" role="tabpanel" aria-labelledby="profile-tab">
      <p>Hola pestaña PROFILE</p>
      </div>
      <div class="tab-pane fade" id="contact" role="tabpanel" aria-labelledby="contact-tab">
      <p>Hola pestaña CONTACT</p>
      </div>
    </div>
    `
